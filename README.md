# POSTER-3D — by Abhirup Payne

AI-Powered 3D Poster Studio. 27 themes, 12 layouts, drag logo, export PNG/JPG.

## How to Deploy on GitHub Pages

### Step 1 — Create a new GitHub repository
- Go to https://github.com/new
- Name it anything e.g. `poster3d`
- Set it to **Public**
- Do NOT add README, .gitignore or license (keep it empty)
- Click **Create repository**

### Step 2 — Upload the files
- Click **uploading an existing file** on the repo page
- Drag and drop ALL files from this ZIP (index.html, README.md, .nojekyll, and the .github folder)
- Click **Commit changes**

### Step 3 — Enable GitHub Pages
- Go to your repo **Settings** tab
- Click **Pages** in the left sidebar
- Under **Source** select **GitHub Actions**
- The workflow will run automatically
- Wait ~60 seconds then visit: `https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/`

## That's it!
